# bus-subscription-system
CSCE 1102 project

Instructions on how to run the application:-
1. Clone the repo
2. Navigate to the repo using "cd ~/bus-subscription-system"
3. Run the command "mdkir build && cd build"
4. Run the command "cmake .. -DCMAKE_BUILD_TYPE=Debug"
5. Run the command "make -j${nproc}
6. In the first terminal, make sure you are in the directory ~/bus-subscription-system/build and run the command "./bin/bus_server" to run the server
7. In the second terminal, make sure you are in the directory ~/bus-subscription-system/build and run the command "./bin/bus_client" to run the application
